```bash
# 1. Read quick reference (5 min)
cat QUICK_REFERENCE.md

# 2. Get API keys from:
# - https://console.cloud.google.com (Google Places)
# - https://developers.amadeus.com (Amadeus)

# 3. Add to .env file

# 4. Restart backend
cd traveler-backend
node server.js

# 5. Test in browser console
```
